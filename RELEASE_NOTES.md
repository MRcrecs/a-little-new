# Release v0.3.0

## Что вошло

- добавлено клонирование сайтов
- добавлено массовое добавление `path` внутри сайта
- добавлено массовое добавление в раздел `Общий path`
- добавлено избранное для сайтов
- добавлена сортировка по порядку добавления, по имени и с избранными сверху
- сортировка по порядку добавления включена по умолчанию
- фильтр `Все категории` включён по умолчанию при запуске
- исправлена потеря фокуса в поле `Path` при редактировании
- обновлены тесты для новой модели `Site`

## Основные файлы релиза

- `url-auto-opener.exe`
- `README.md`

## Примечания

- формат `sites.json` расширен полем `favorite`
- совместимость со старыми данными сохраняется: отсутствие `favorite` трактуется как `false`

# Release v0.2.0

## Что вошло

- модульная архитектура вместо монолитного `main.py`
- выделены слои `state`, `url_service`, `workers`, `models`, `window`
- добавлены базовые unit-тесты для JSON/state и URL-логики
- сохранена защита от перезаписи битого `sites.json` с резервной копией
- фоновые проверки `path` оставлены вне GUI-потока
- исправлено автозаполнение поля названия сайта при очистке
- обновлена внутренняя документация проекта в `AGENTS.md`

## Основные файлы релиза

- `url-auto-opener.exe`
- `README.md`

## Примечания

- структура проекта изменилась, но формат `sites.json` остался совместимым
- локальная база сайтов по-прежнему хранится отдельно в `sites.json`
- `sites.json` не входит в репозиторий и релизный архив по умолчанию
